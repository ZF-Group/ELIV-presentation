# Security Policy

## Best practices

- Dependabot is activated for this repository.
- Dependency graph is enabled to track dependencies.
- Using environment variables for sensitive data.
- Do not hardcode secrets or private keys.

## Reporting a Vulnerability
If you discover a security vulnerability in this project, please report it responsibly.
Do not create a public issue.
Contact us directly at: ospo@zf.com
Include as much detail as possible to help us reproduce and fix the issue quickly.
We aim to respond in time and will work with you to resolve the issue promptly.

## Disclosure Policy
We follow a responsible disclosure policy.
Please give us time to investigate and fix any vulnerabilities before disclosing them publicly.
